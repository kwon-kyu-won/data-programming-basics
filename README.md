# data-programming-basics

# 머신러닝 기초 실습 (Forge 데이터셋)

이 프로젝트는 `mglearn` 라이브러리를 활용하여 데이터를 시각화하고, 
`scikit-learn`을 이용해 데이터를 나누는 기초 과정을 포함하고 있습니다.

## 🛠 환경 설정
이 프로젝트는 가상환경에서 작성되었습니다. 아래 명령어로 필요한 라이브러리를 설치할 수 있습니다.
`pip install -r requirements.txt`

## 📝 주요 학습 내용
1. **데이터 시각화**: `make_forge()` 데이터셋의 특성 파악
2. **데이터 분할**: `train_test_split`의 다양한 옵션(`test_size`, `stratify`) 실습
